# Meetgo — обзор продукта

## Что это

Meetgo — мобильное приложение для спонтанных знакомств и встреч в реальной жизни.

В отличие от классических дейтинг-сервисов, приложение строится вокруг **активностей, а не анкет**. Пользователь публикует намерение что-то сделать в ближайшее время: выпить кофе, погулять, сходить в бар, посетить выставку, поработать из кофейни и т.д.

**Основная идея** — помочь людям найти компанию для конкретного совместного опыта здесь и сейчас.

## Философия

- реальные встречи важнее переписки;
- активность важнее профиля;
- действие важнее потребления контента;
- простота важнее количества функций;
- продукт должен мотивировать людей выходить в реальный мир, а не проводить время внутри приложения.

## Стадия

Проект на стадии **MVP**. Главная цель — максимально быстро проверить продуктовую гипотезу и получить обратную связь от реальных пользователей.

## Принцип разработки

При проектировании и разработке отдаём предпочтение **простым решениям**:

- избегаем преждевременной оптимизации;
- не добавляем функциональность, которая не требуется для проверки основной идеи продукта;
- каждое архитектурное усложнение должно оправдываться текущей потребностью MVP, а не гипотетическим будущим.

> Этот принцип — рабочий критерий при ревью: если фича/абстракция не помогает проверить гипотезу «люди встречаются офлайн через активности» — она пока не нужна.
