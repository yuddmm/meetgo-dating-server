# Справочник тегов встреч (seed)

Стартовый набор **тегов встреч** для MVP. В отличие от [интересов](./interests.md) (которые описывают «кто я»), теги встреч описывают **что сделать вместе здесь и сейчас** — это намерение анкеты `MeetingAd` (см. [matching.md](./matching.md)).

Это **НСИ** единого типа `{ id, value }` (см. [api.md](./api.md) → «Соглашения»): `value` — стабильный ключ в `SCREAMING_SNAKE_CASE`, перевод — на клиенте по i18n. Бэкенд **сидит** этот список в миграции и отдаёт в контракте модуля встреч (эндпоинт — TBD, добавим вместе с модулем встреч).

- `key` — стабильный ключ (`value` в НСИ), **не меняется** после релиза.
- `ru` — русский перевод для клиентского i18n (передаётся в репозиторий фронта).
- Это **отдельный справочник** от интересов — одинаковые строковые ключи в двух таблицах допустимы (разные сущности, разные `id`).

> ~50 ключей для старта. Расширяется через сид/админку; **ключи не переименовываем** — только добавляем (на них ссылаются анкеты).
