## Статус

- **Фронтенд** — инициализирован (Expo SDK 54, React Navigation). Этот репозиторий.
- **Бэкенд** (Go) — реализованы auth и профиль/онбординг (включая фото). Контракт в [api.md](./api.md) актуален; живой источник схем — Swagger (`/swagger/index.html`). Модуль встреч/чатов — впереди.

> Доки `overview` / `frontend` / `conventions` описывают **фактическое** состояние.
> `backend` и часть `architecture` — это **проектные решения и открытые вопросы**, помечены `🚧 TBD`.
