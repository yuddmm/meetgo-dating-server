# Meetgo — документация

Документация проекта Meetgo. Держим её короткой и актуальной — по принципу продукта «простота важнее количества».

## Содержание

| Документ                       | Статус  | О чём                                                                        |
| ------------------------------ | ------- | ---------------------------------------------------------------------------- |
| [overview.md](./overview.md)   | ✅      | Продукт: идея, философия, стадия, принципы                                   |
| [auth-flow.md](./auth-flow.md) | ✅      | Флоу авторизации и онбординга (email + OTP, создание профиля)                |
| [units.md](./units.md)         | ✅      | Единицы измерения (рост/вес): СИ в БД, конверсия на фронте                   |
| [matching.md](./matching.md)   | ✅      | Логика мэтчинга: роли, отклики/кандидаты, временные/постоянные чаты, premium |
| [api.md](./api.md)             | ✅      | Контракт фронт↔бэк: auth (OTP, токены), профиль, фото, обработка 401         |
| [tasks.md](./tasks.md)         | 🚧      | Бэклог фронта: ближайшие экраны и задачи                                     |
| architecture.md                | 🚧 план | Общая архитектура: RN-клиент + Go-бэкенд (модульный монолит)                 |

| backend.md                     | 🚧 план | Go-бэкенд: решения и открытые вопросы (передаётся в Go-репозиторий)          |
| conventions.md                 | 🚧 план | Код-стайл, линт, формат, импорты, алиасы                                     |

## Статус

- **Фронтенд** — инициализирован (Expo SDK 54, React Navigation). Этот репозиторий.
- **Бэкенд** (Go) — реализованы auth и профиль/онбординг (включая фото). Контракт в [api.md](./api.md) актуален; живой источник схем — Swagger (`/swagger/index.html`). Модуль встреч/чатов — впереди.

> Доки `overview` / `frontend` / `conventions` описывают **фактическое** состояние.
> `backend` и часть `architecture` — это **проектные решения и открытые вопросы**, помечены `🚧 TBD`.
